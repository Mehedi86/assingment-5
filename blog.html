Important FAQ

What is the Document Object Model (DOM)?

It is a model based on javascript object as like a tree. Its convert every html element and its inner child or things as node and can provide some useful facility. Facilities it provide like manipulate the website or html and also its css style, select or get the html element, add event etc.

How do you select an element from the DOM?

If I want to select an element I will use method like getElementById(), querySelector().

What is the event delegation in the context of the DOM, and why is it useful?

Event delegation is a concept like that, if parent element has event handler or event listener then, this event listener also work on this parent's child. We can use event listener on multiple child element just use a event listener on their parent. It is very useful because it provide the scope to use less event listener and it decrease the complexiety of code.

How do you manipulate an elements attributes and styles using the DOM?

First I get the element and then manipulate the attribute and styles by the use of simultenuosly setAttruibute(), getAttribute(), removeAttribute(), style. As example,
For manipulate attribute:
document.getElementById('div-element').getAttruibute('class');
document.getElementById('div-element').setAttruibute('class' 'w-full');
For manipulte styles :
document.getElementById('div-element').style.color = 'green';
